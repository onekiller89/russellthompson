# Russell Thompson - Personal Portfolio

Personal portfolio site for Russell Thompson, Chief Engineer & Automation Lead.

**Live site:** [russellthompson.pages.dev](https://russellthompson.pages.dev/)

## Tech Stack

- Static HTML/CSS/JS (no build step)
- Hosted on Cloudflare Pages with GitHub auto-deploy
- Custom particle animation, scroll-triggered sections, responsive design

## Deployment

Pushes to `main` automatically deploy to Cloudflare Pages. No build command required.
